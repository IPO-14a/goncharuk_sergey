MYT(Goncharuk_Sergey)

Разрабатываемое ПО позволяет управлять личным временем путём создания заметок, списков дел, отслеживания заметок по карте.

AngularJS, IonicJS, Google Maps API, Bootstrap CSS

Abstract

Функционал программы позволяет создавать, редактировать и удалять заметки, расписание, отметки на карте, приглашать друзей на мероприятия.

Экплуатационное назначение - управление личным временем и повседневными делами.

Потребность в разработке нового решения при существовании готовых продуктов, которые частично реализуют требования, описанные в ТЗ, обусловлено тем, что обычно предлагаемый функционал содержится одновременно в несколькиих программных продуктах. В большинстве случаев интерфейс подобного ПО не продуман и сделан "на скорую руку".

Пример №1 - Добавление новой заметки

Пользователь добавляет новую заметку в записную книгу.

$scope.addNote = function(){

$scope.notes.push({

title: $scope.noteTitle,

text: $scope.noteText,

pubdate: date = new Date(),

// pubday: day = date.getDay()

});

$scope.noteTitle = "";

$scope.noteText = "";

}

<form role="form" class="form-inline">

<div class="form-group">

<label for="email">Заголовок</label>

<input type="text" class="form-control" ng-model="noteTitle" placeholder="Введите заголовок">

</div>

<div class="form-group">

<label for="text">Текст заметки</label>

<input type="text" class="form-control" ng-model="noteText" placeholder="Введите заметку">

</div>

<button type="submit" class="btn btn-success" ng-click="addNote()">+</button> </form>

Пример № 2 - Удаление существующей заметки

Пользователь удаляет существующую заметку из записной книги.

$scope.delNote = function(index){

$scope.notes.splice(index, 1);

}

<button class="btn btn-danger btn-close" ng-click="delNote($index)">x</button>
